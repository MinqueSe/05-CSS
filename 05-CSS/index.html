Background-color

Background-color es un propiedad de CSS que define el color de fondo de un elemento, puede ser el valor de un color o la palabra "transparente"


Ejemplo: background-color: white;


background-image

La propiedad CSS background-image establece una o más imágenes de fondo en un elemento

Ejemplo: background-image: url (assets/carpincho.jpg)


La palabra quedaria arriba de la imagen


color

La propiedad color CSS establece el valor de color de primer plano del texto y las decoraciones de texto de un elemento,
y establece el valor de color actual. currentcolor puede usarse como un valor indirecto en otras propiedades
y es el valor predeterminado para otras propiedades de color, como border-color.


Ejemplo: color: red;


text-aling

La propiedad CSS de alineación de texto establece la alineación horizontal del contenido de nivel en línea dentro de un elemento de bloque o cuadro de celda de tabla.
Esto significa que funciona como alineación vertical pero en dirección horizontal.


Ejemplo: text-align: center;

text-align: end;

text-align: start;


width

La propiedad CSS ancho establece el ancho de un elemento. De forma predeterminada, establece el ancho del
área de contenido, pero si el tamaño del cuadro se establece en border-box, establece el ancho del área del borde.


Ejemplo img src="/assets/carpinchoberto.jpeg" alt="carpincho" width="300p"


Height

La propiedad CSS de altura especifica la altura de un elemento. De forma predeterminada, la propiedad define
la altura del área de contenido. Sin embargo, si el tamaño del cuadro se establece en cuadro de borde, en su
lugar determina la altura del área del borde.


Ejemplo img src="/assets/carpinchoberto.jpeg" alt="carpincho" width="300p" height="100p"


Border

La propiedad CSS abreviada de borde establece el borde de un elemento. Establece los valores de border-width, border-style y border-color


Ejemplo
border: solid;


--Border-Collapse--

La propiedad CSS border-collapse establece si las celdas dentro de una tabla tienen bordes compartidos o separados
